# Quick Reference — Example prompts for every module

Open your **Avi Snow OS** project at claude.ai, start a new chat, and type any of these. Tweak the details.

---

## caption — social post

```
caption: just played Sound, closed with a melodic techno edit of a Nigerian vocal I'd been sitting on for six months. [friend] was on before me. Room was locked.
```

**Variants:**
```
caption for tiktok: studio today, working on a 122 bpm afro-house thing. still figuring out the top line.
```

```
caption for x: the moment a crowd locks into a track you made two weeks ago at 3am in your bedroom is not a normal experience and I don't want to get used to it
```

```
caption, 2 variants, more introspective tone: six months in on endless dream records. learning what signing an artist actually means.
```

## General modifiers you can add to any prompt

- `variants: 3` — produce 3 full sets (default varies by module)
- `notes: "drier tone"` or `notes: "lean more introspective"` — steering for this call
- `platform: ig` / `platform: tiktok` / `platform: x` — for captions
- `word count: 150` — for EPK or newsletter
- `tone: casual` / `tone: professional` — override default voice for this one call

---

## Examples of steering mid-conversation

If the draft isn't quite right, reply in the same chat:

```
too marketer-y, try again. drop the last line. more specific about the BPM.
```

```
the short version is great, redo medium and long, tighter rhythm.
```

```
replace "journey" with literally anything else, it's banned.
```

The project picks up steering naturally — treat it like texting a collaborator.
